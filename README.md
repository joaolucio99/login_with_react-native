# Login React Native

Desenvolvido para Android

Utilização:

    Android Studio:
        instalar as dependências com "npm i"
        abrir o app com "npx react-native run-android"
    Via apk:
        instalar o arquivo apk disponibilizado na pasta Instalador da raíz do projeto
